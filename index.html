Erika Cordeiro - Desenvolvedora Web JavaScript

Biografia

Meu nome é Erika, e sou uma entusiasta apaixonada por tecnologia e desenvolvimento de software. Com uma sólida formação em lógica de programação, adentrei no vasto mundo da programação web com ênfase em JavaScript, onde explorei as mais modernas tecnologias e técnicas para criar experiências digitais envolventes e dinâmicas. Além disso, mergulhei no universo do Python, especialmente focado no framework Django, aprofundando-me na construção de aplicativos web robustos e escaláveis. Atualmente, estou imerso no Bootcamp de Desenvolvimento de Jogos do IFOOS, onde estou aprimorando minhas habilidades na criação de jogos usando JavaScript. Esta jornada tem sido emocionante, pois combina minha paixão pela programação com minha criatividade e interesse em jogos. Paralelamente aos meus estudos técnicos, estou dedicando tempo ao aprimoramento do meu inglês, reconhecendo a importância dessa habilidade no cenário global da tecnologia. Minha abordagem proativa e minha paixão por aprender me capacitam a enfrentar desafios de forma eficaz, enquanto minha natureza comunicativa e minha habilidade em trabalhar em equipe garantem que eu possa colaborar harmoniosamente em qualquer ambiente de trabalho. Estou ansioso para continuar minha jornada de crescimento profissional, explorando novas tecnologias e contribuindo para projetos empolgantes que impulsionam a inovação e transformam ideias em realidade.

Projetos

Descrição do Projeto: Página Trilha de CSS da DIO

Trilha de CSS da DIO

A página de Trilha de CSS da DIO é um projeto desenvolvido com HTML e CSS, oferecendo uma experiência visualmente envolvente e interativa. Com um design moderno e responsivo, a página guia os participantes em uma jornada de aprendizado imersiva, desde os conceitos básicos até as técnicas avançadas de estilização web. Ao fornecer acesso à comunidade, recursos adicionais e oportunidades profissionais, o projeto visa capacitar os participantes a se tornarem especialistas em CSS e líderes no desenvolvimento web.


Descrição do Projeto: Projeto Create React App

Template React

Este projeto oferece uma estrutura inicial para desenvolvimento de aplicações React. Ele facilita a configuração do ambiente de desenvolvimento, permitindo que os desenvolvedores iniciem rapidamente a criação de aplicativos React sem se preocupar com configurações iniciais complexas.


Descrição do Projeto: Primeiro Jogo 2D na Godot

Patito's Quest: Em busca da Pata Perdida

Embarque em uma aventura encantadora neste jogo simples, onde um patinho embarca em uma jornada para encontrar sua pata perdida, enfrentando desafios emocionantes em um percurso repleto de obstáculos de espinhos. Desenvolvido como um projeto inicial na Godot Engine, este jogo apresenta conceitos fundamentais de jogabilidade, detecção de colisões e navegação de personagem, proporcionando uma experiência divertida e educativa para jogadores de todas as idades.
